## 加盟 vs 直营识别

### 硬验证方法(优先用, 最可靠)

按可靠性从高到低, 能用就用:

| 方法 | 可靠性 | 操作 | 加盟信号 |
|------|--------|------|----------|
| **品牌官网门店列表** | 🟢最高 | 万豪/洲际/温德姆/华住官网"查找酒店"搜该店 | 官网无此店 = 加盟挂牌 |
| **会员APP能订+累积积分** | 🟢高 | 万豪/IHG/华住APP搜该店, 能订且积分 | 不能积分/APP搜不到 = 挂牌 |
| **小红书"未上官网/叫另一个名"** | 🟢高 | 搜"{酒店名} 加盟 翻牌 改名" | "原XX酒店改名""没上官网" = 挂牌(D3英迪格实证) |
| **工商信息(企查查)** | 🟡中 | 查运营方主体 | 运营方≠品牌方 = 加盟 |

### 间接识别线索(辅助, 不单独定论)

| 线索 | 加盟特征 | 直营特征 |
|------|----------|----------|
| **历史名称** | 有改名记录（如"凤凰花酒店"→"蔚景温德姆"） | 品牌名稳定 |
| **差评关键词** | "服务跟不上"、"和官网图片不符"、"管理混乱"、"挂牌卖质"、"设施陈旧但标新" | "排队久"、"价格贵"、"停车远" |
| **评价一致性** | 同品牌不同店评分差异大（4.2 vs 4.8） | 同品牌评分稳定 |
| **设施描述** | 设施与品牌标准不符 | 设施描述与品牌标准一致 |
| **开业 vs 翻牌** | 翻牌店开业时间短但设施旧 | 新开业设施全新 |

### 判断流程

```
收到酒店信息
  │
  ├─ ① 硬验证(优先): 品牌官网门店列表 / 会员APP能否订+积分
  │   ├─ 官网无/不能积分 → 确认加盟挂牌, 降信任+拉黑
  │   └─ 官网有/能积分 → 直营或授权直营, 正常展示
  │
  ├─ ② 硬验证不可行时, 用间接线索+小红书:
  │   ├─ 检查历史改名 → 有 = 加盟翻牌嫌疑
  │   ├─ 检查差评关键词 → 匹配加盟特征词 = 高风险
  │   ├─ 检查同品牌其他店评分一致性 → 差异大 = 加盟风险
  │   └─ 小红书搜"未上官网/叫另一个名" → 命中 = 挂牌
  │
  └─ 综合判断:
      ├─ 确认加盟翻牌 → 标注⚠️, 降信任等级
      ├─ 疑似加盟 → 标注❓, 建议用户确认
      └─ 直营/不确定 → 正常展示
```

### 加盟翻牌处理

1. 该酒店 → 拉黑 + 写入 blacklist.md
2. 同品牌同城同级别 → 降一级信任等级
3. 如果该品牌已多次出现加盟问题 → 整个品牌线降级

**示例**:
```
凤凰蔚景温德姆(加盟翻牌) 体验差
  → 拉黑该店
  → 温德姆4星级全线降为🟠低信任
  → 后续仅允许直营戴斯(Days)
```

## 黑榜系统

**存储位置**: `~/.trip-scout/blacklist.md`

### 黑名单格式

```markdown
# 酒店黑名单

## 拉黑规则
- 用户明确说"再也不住" → 永久拉黑
- 本质性差评(卫生/安全/欺诈) → 拉黑 + 通知用户
- 加盟翻牌且体验差 → 拉黑该店 + 同品牌同城降级

## 黑名单
| 酒店 | 品牌 | 城市 | 拉黑原因 | 拉黑日期 | 范围 |
|------|------|------|----------|----------|------|

## 品牌降级记录
| 品牌 | 原等级 | 降级原因 | 新等级 | 影响范围 |
|------|--------|----------|--------|----------|
| 温德姆4星级 | 🟡中 | 凤凰蔚景加盟翻牌差体验 | 🟠低 | 仅允许直营戴斯 |
```

### 黑名单匹配规则

搜索结果中的酒店，按以下顺序匹配黑名单：

1. **精确匹配**: 酒店名完全相同 → 直接移除
2. **品牌+城市匹配**: 同品牌+同城市+同级别 → 移除并标注
3. **品牌降级匹配**: 品牌已被降级 → 标注⚠️, 需额外验证

## 信任等级动态调整

### 升级条件

| 条件 | 操作 |
|------|------|
| 入住后用户说"不错/很好/推荐" | 品牌信任 +1 (上限: 当前等级) |
| 同品牌3次入住无差评 | 品牌信任 +1 |
| 发现新的直营好店 | 记录正面经验 |

### 降级条件

| 条件 | 操作 |
|------|------|
| 入住后用户说"再也不住" | 拉黑该店 + 品牌信任 -1 |
| 发现本质性差评 | 拉黑该店 + 品牌信任 -1 |
| 发现加盟翻牌差体验 | 拉黑该店 + 品牌信任 -2 |
| 同品牌2次以上加盟问题 | 品牌线整体降级 |

### 降级后果

```
🟢高 → 🟡中: 不再默认推荐, 需看具体店评分
🟡中 → 🟠低: 需额外验证, 标注⚠️
🟠低 → ⚫黑: 整个品牌线拉黑(极端情况)
```
