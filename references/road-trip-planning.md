# 自驾游行程规划方法论

> 基于 `trip-planning.md` 通用方法论，叠加自驾游特有的决策维度。通用部分（四拍交互、选点/选餐原则、信息源优先级）不重复，只写自驾增量。

---

## 一、自驾游 vs 通用出行的差异清单

| 维度 | 通用出行 | 自驾游增量 |
|------|---------|-----------|
| 交通 | 地铁/公交/打车，时效可控 | 自驾，时效受路况/天气/节假日车流影响，里程和时长是硬约束 |
| 车辆 | 无 | 取车/还车节点、车型选择、儿童座椅、全险、异地还车费 |
| 路线 | 地铁线路唯一选择 | 高速/国道/省道多选，路况、区间测速、冬季封路需评估 |
| 驾驶节奏 | 步行5km算极限 | 每天驾驶不宜超4-5h（亲子）/6-7h（纯成人），长途日必须配轻量游玩 |
| 费用 | 交通费固定 | 油费/过路费/租车费/轮渡费/异地还车费，高速免费时段可利用 |
| 住宿 | 酒店位置决定区域 | 住宿必须考虑停车场（免费停车优先）、取车方便性 |
| 车队 | 单人/双人 | 双家庭多车协作：同步出发、AA分摊、通信方式 |
| 风险 | 天气/预约 | 路况（高原山路/雨雪/塌方）、夜间山路、新能源补能、轮渡停航 |

## 三、自驾游判断顺序（在通用判断顺序基础上插入）

通用判断顺序为：天数够不够 → 酒店在哪 → 用户讨厌什么 → 天气 → 预约 → 节假日 → 补餐厅

**自驾游插入额外步骤**：

1. 天数够不够
2. **取车/还车节点在哪**（决定首尾天路线）
3. **总驾驶里程是否超限**（亲子每天≤300km/4h，纯成人≤500km/6h）
4. ⚠️ **风险前置：最大变数路段排最前面**（见下方"风险前置原则"）
5. ⚠️ **预约通行检查**（独库公路北段等需预约路段，见下方"预约通行与封路通知获取"）— 抢不到预约=无法通行，比封路更致命
6. 酒店在哪（优先免费停车、取车方便）
7. 用户讨厌什么
8. **路线选择：高速 vs 国道 vs 省道**（风景、路况、时间三角权衡）
9. 哪些点看天气
10. 哪些点需要预约（景区门票预约 + **路段预约通行**）
11. 哪些点会在节假日爆炸
12. **驾驶日不能排太满的景点**（长途日配轻量游玩，短途日可多排）
13. 最后才补餐厅和咖啡

## 四、路线选择原则

### 路线三角权衡

每段路线需在 **时间 / 路况 / 风景** 三角中做取舍：

| 优先级 | 典型场景 | 选择倾向 |
|--------|---------|---------|
| 时间优先 | 长途赶路日、最后一天返程 | 全高速，不绕路 |
| 风景优先 | 当天驾驶量轻、用户明确想看沿途风光 | 国道/省道（如217省道、G227最美国道） |
| 路况优先 | 亲子出行、雨雪天、夜间驾驶 | 避开省道山路，选高速或路况稳定的国道 |

### 路线评估必查项

- **区间测速**：川西、新疆等旅游区大量区间测速，必须遵守限速
- **冬季封路/塌方**：高原省道冬季可能封路，出发前查当地交通公众号
- **夜间山路**：绝不安排夜间走山路（弯道多、无路灯、动物出没）
- **炮弹坑路**：部分省道/县道路况极差（如丹巴→玉科大草原112km/3.5h炮弹坑路），亲子出行直接排除
- **服务区质量**：热门线路服务区可能爆满（天全服务区节假日不建议吃饭）

### 路线数据来源

- **高德地图驾车路线规划API**：里程、时长、过路费逐段校正
- **当地交通公众号/微博**：封路、塌方实时通知
- **小红书近期游记**：路况实况、测速提醒（不可用时用互联网搜索替代）

### 风险前置原则（实战核心策略）

> **把最大变数放在行程最前面，而不是最后面。** 封路/塌方/天气等不可控因素，越早遇到越有时间调整。

**为什么风险要前置**：
- 独库公路等季节性道路可能封路，如果放在行程最后1-2天，封路时只剩1天调整空间，几乎无法补救
- 反向游玩：先走独库公路（D2独山子→唐布拉），封路了还有10天调整（改走赛里木湖方向或绕行伊宁）
- 放在最后：封路了只能临时取消或绕远路赶回，行程崩溃

**路线方向选择**：
- 有季节性封路风险路段 → **反向游玩**，先走风险路段
- 有天气敏感路段（高原垭口） → 优先安排在天气窗口期
- 有预约瓶颈景点 → 尽早安排，避免后期约满

**风险评估模板**（路线推荐时必填）：

| 风险路段 | 风险类型 | 发生概率 | 影响程度 | 建议位置 | 降级方案 |
|---------|---------|---------|---------|---------|---------|
| 独库公路北段 | **预约通行+季节性封路** | **高（旺季7-9月）** | **致命（未预约罚款+扣分，无法通行）** | 行程前段(D2-D3) | 抢不到预约→绕行伊宁+赛里木湖，多走1天；封路→同前 |
| 独库公路中/南段 | 季节性封路 | 中-高 | 致命（无替代路） | 紧接北段后 | 绕行伊宁+赛里木湖，多走1天 |
| 昭苏→夏塔 | 山路雨雾 | 中 | 中（可等天晴） | 当天上午 | 改去昭苏天马园 |
| 高原垭口 | 大雪封路 | 低（9月） | 高 | 上午通过 | 等下午化雪 |

**动态调整能力**（行中）：
- 出发前查封路状态（当地交通公众号/高德App实时路况）
- 行中遇到封路 → 立即切换降级方案，不硬闯
- 降级方案在路线推荐阶段就预设好，行中直接查表执行
- 通知同行家庭同步调整（双家庭协作）

## 五、驾驶节奏规划

### 每日驾驶上限

| 人群 | 每天最大驾驶时长 | 每天最大里程 | 说明                     |
|------|-----------------|-------------|------------------------|
| 亲子（含6岁儿童） | ≤4-5小时 | ≤300km | 不赶早10点出发，儿童在车上不超过5h |
| 纯成人 | ≤6-7小时 | ≤500km | 超过7h疲劳驾驶风险急剧上升         |
| 高原山路 | 降低30% | 降低30% | 弯道多、海拔高、注意力消耗大         |

### 驾驶日配景点原则

- **长途日（≥3h驾驶）**：只配1个轻量景点 + 1顿顺路餐，当天核心任务是到达目的地
- **短途日（≤1.5h驾驶）**：可排2-3个景点 + 自由时间
- **纯驾驶日**：只排路线亮点（沿途观景台、服务区美食），不排正式景点
- **第一天**：取车 + 轻量适应（≤2h驾驶），不跑远
- **最后一天**：还车 + 返程，不排景点（除非还车点就在景区附近）

### 飞行抵达日住宿规则（强制）

**D1必须住飞行目的地城市**，不安排当天开车赶往下一站。原因：
1. 飞行+取车+午餐+熟悉车辆，大半天已过，再开3h+太赶且疲劳驾驶风险高
2. D1住飞行目的地（如乌鲁木齐），D2早上租车出发，行程从容
3. 机票时段选择更多——不用抢早班赶路，中下午航班选择更多且可能更便宜
4. D1住大城市的好处：酒店选择多、餐饮便利、可采购旅途物资

**行程表调整**：D1原定目的地（如独山子）变为D2的第一站，整体天数不变，只是D1从"赶路日"变为"抵达+取车+适应日"

### 出发时间偏好（亲子）

- 亲子自驾默认 **9:30出发**，拒绝早起
- 唯一例外：核心景点需要开门就进（如七彩丹霞看晨光），可安排7:30出发但前一晚必须早睡
- 每天写清"几点出发"，不做模糊表述

## 六、车辆与租车规划

### 车型选择优先级

| 场景 | 推荐车型 | 原因 |
|------|---------|------|
| 双家庭6人 | 7座MPV（丰田格瑞维亚/岚图梦想家） | 两家坐一车省钱省心，空间大儿童友好 |
| 双家庭多车 | 各开一辆5座SUV | 灵活度高，AA分摊简单，但油费翻倍 |
| 3-4人 | 5座SUV或轿车 | 够用，油耗低 |
| 高原/山路 | SUV（底盘高、抓地力强） | 省道县道路况保障 |

### 租车关键项

- **全险必买**：自驾游路况不可控，磕碰概率高
- **儿童安全座椅**：租车平台可租（一嗨/神州），提前APP锁定+电话确认门店库存
- **异地还车费**：计算进总预算（如海口取→湛江还，一嗨3995元含异地费）
- **油量规则**：满油取满油还（提前在还车点附近加满油）
- **车型锁定**：提前3天APP锁定+电话确认，避免到店无车

### 异地还车策略（不走回头路）

> **环线自驾游的取还车节点决定首尾天路线，是路线规划的核心约束之一。**

**原则**：取车点和还车点尽量不同，避免走回头路浪费1-2天。需计算异地还车费 vs 节省的油费+过路费+时间。

**决策模板**：

| 方案 | 取车 | 还车 | 异地还车费 | 节省里程 | 节省时间 | 推荐 |
|------|------|------|-----------|---------|---------|------|
| 环线回原点 | 乌鲁木齐 | 乌鲁木齐 | ¥0 | 0km | 0天 | 走回头路 |
| 异地还车 | 乌鲁木齐 | 伊宁 | ¥X | ~580km | 1-2天 | ✅ 不走回头路 |

**异地还车可行性检查**：
1. 还车城市是否有机场且有直飞/经停航班回出发城市
2. 租车平台是否支持该城市异地还车（一嗨/神州覆盖范围）
3. **异地还车费**：一嗨租车永久免异地还车费，神州租车限时免除（关注活动），**通常可忽略此项**
4. 机票价格对比：同时间段直飞价格（排除早6点/晚22点等极端时段的低价票），加上额外住宿费（如还车城市距机场>4h需住一晚）
5. 整体费用对比：机票+住宿+油费+过路费，如异地还车方案每家庭多花>¥800-1000则选原点还车

**路线规划时的机场选项**：
- 列出路线沿途所有机场（高德POI搜索"机场"）
- 计算每个机场到最近路线节点的距离
- 评估机场航班覆盖（直飞/经停/无）
- 输出异地还车方案供用户选择

### 车队协作（双家庭）

- **出发同步**：约定统一出发时间，途中保持微信/电话联络
- **AA分摊**：租车费、油费、过路费按家庭分摊；餐饮/门票按人头
- **轮渡等特殊场景**：提前统一购票，升舱统一决策（五星船咖啡厅200元/6人）
- **导航方式**：各车独立导航，不依赖跟车（避免走错路口掉队）

## 八B、预约通行与封路通知获取

> **2026年6月25日起，独库公路北段正式实施分时段预约通行制度。7月10日起严查处罚，未预约或错时段通行将面临罚款200元并记1分。** 这是自驾游新增的致命级风险——不是"可能堵车"，而是"不预约就不让走"。

### 哪些路段需要预约

目前仅**独库公路北段**（乌苏驿 — 乔尔玛 — 那拉提，约185公里）需要预约。中段（那拉提往巴音布鲁克方向）和南段（往库车方向）无需预约。

其他热门自驾路线（川西318/317、青海湖环线等）目前无需预约，但政策可能扩展，出发前务必确认。

### 预约核心规则

| 规则 | 内容 |
|------|------|
| 通行时间 | 每日 8:00 - 19:00（社会车辆）；19:00后全线封闭施工，禁止驶入，且必须在19:00前驶出山区 |
| 预约时段 | 5个时段，每2小时一批：8:00-10:00 / 10:00-12:00 / 12:00-14:00 / 14:00-16:00 / 16:00-19:00 |
| 适用车型 | 仅7座及以下小型客车、5吨及以下轻型货车；8座以上客车、拖挂房车、危化品车、挂车等全线禁行 |
| 放号规则 | 可提前1-7天（含当天）预约；每天凌晨0:00释放第7天名额；旺季热门时段非常紧张，建议定闹钟抢号 |
| 费用 | 全程免费 |

### 预约渠道与操作流程

**预约平台**（任选其一）：
- "游新疆一码游"微信小程序（推荐，最便捷）
- "平安天山""新疆交警""新疆路网""新疆发布"微信公众号
- "石榴云""丝路视听"客户端

**操作流程**：
1. 登录平台 → 选择"自驾预约"
2. 填写车牌号、驾驶人信息、手机号
3. 选择出行日期、预约入口（乌苏驿/乔尔玛/那拉提）、时段
4. 确认提交（全程免费）

### 避坑与违约指南

| 场景 | 规则 | 说明 |
|------|------|------|
| 迟到补救 | 可在"预约记录"申请延后1次，系统自动延长1小时 | 不视为违约，但每车每次预约仅可申请1次 |
| 二次进入 | 当天预约成功进过一次北段，中途驶出后当天再次进入 | 无需二次预约 |
| 取消 | 必须在预约时段开始前取消 | 单日取消满3次，当日无法继续预约 |
| 爽约 | 未通行且未取消 | 全年累计爽约超3次，取消当年预约资格 |
| 不可抗力 | 恶劣天气/道路塌方/交通管制导致无法通行 | 可免费取消，不计入爽约次数 |

### 封路/路况实时通知获取

**出发前必查**（按优先级排序）：

1. **"新疆路网"微信公众号** — 最权威的实时路况和封路通知
2. **"新疆交警"微信公众号** — 交通管制、天气预警、事故通报
3. **高德App实时路况** — 路线规划时查看当前路况（红色=拥堵/封闭）
4. **小红书近期游记** — 搜索"独库公路 路况"获取最新实况（1-3天内最准，不可用时用互联网搜索替代）

**行中动态调整**：
- 出发前1天晚上查封路状态 → 确认预约时段是否仍可通行
- 出发当天早上再查一次 → 山区气候多变，可能临时管制
- 遇封路 → 立即切换降级方案，不硬闯
- 降级方案在路线推荐阶段就预设好，行中直接查表执行

### 预约通行对行程规划的影响

1. **风险前置原则强化**：有预约通行要求的路段，必须排在行程前段。抢不到预约→还有时间调整；排在最后→封路/约满时行程崩溃
2. **出发时间倒推**：预约了8:00-10:00时段，意味着必须7:30前到达入口，倒推住宿点必须在入口附近（如独山子/奎屯住一晚）
3. **错峰策略**：抢不到上午热门时段，选16:00-19:00（名额充足+雪山夕阳美景）
4. **行程弹性**：预约通行路段当天不排紧凑衔接，留1-2小时缓冲
5. **加油补给**：沿途加油站少，在独山子/乔尔玛/那拉提看到油量剩一半就加满

### 自驾游技能中的预约检查流程

在路线推荐阶段（SKILL.md 场景二 Phase 1），风险评估步骤必须包含：

```
预约通行检查清单：
□ 路线是否经过需要预约的路段？（独库公路北段等）
□ 预约时段是否与行程时间匹配？
□ 住宿点是否在预约入口附近？（确保能按时到达）
□ 是否已标注预约平台和操作流程？
□ 降级方案是否已预设？（抢不到预约/封路时的替代路线）
□ 出发前1天+当天早上的封路确认流程是否已写入攻略？
```

## 九、住宿选择增量约束

自驾住宿除了通用考量（位置、品质），追加：

- **停车场**：必须有，免费优先（市区酒店停车场可能¥50-100/晚）
- **取车方便**：如租车，酒店离取车/还车点越近越好
- **车位宽度**：7座MPV/SUV需要更宽车位，老城区小酒店可能进不去
- **充电桩**：新能源需确认酒店是否有充电设施

## 十二、自驾游常见坑（在通用常见坑基础上追加）

| 常见坑 | 说明 |
|--------|------|
| **不预约就走独库公路北段** | **2026年6月25日起必须预约，未预约罚款200元+记1分，且无法通行** |
| **不查封路/管制通知** | 山区气候多变，出发前1天+当天早上必须查"新疆路网"公众号确认路况 |
| 不看每日驾驶上限 | 亲子一天开6h+，孩子崩溃大人疲惫 |
| 安排夜间山路 | 弯道+无灯+动物=极度危险 |
| 不查区间测速 | 旅游区测速密集，罚款+扣分 |
| 不估算油费/过路费 | 自驾费用大头，低估导致预算爆表 |
| 不确认停车场 | 市区老酒店无车位，7座MPV进不去 |
| 忽略高速免费时段 | 五一/国庆/清明/春节高速免费，可省大量过路费 |
| 不提前锁定车型 | 到店发现没车，被迫换车型或延误 |
| 不备离线地图 | 高原/山区信号差，导航断线 |
| 不看轮渡/航班衔接 | 轮渡停航或航班延误导致整天行程报废 |
| 把长途驾驶日排满景点 | 长途日只能配轻量活动，否则赶不到目的地 |
| 不标路线路况 | 用户不知道某段是山路/省道/测速路段，驾驶时措手不及 |

## 十三、自驾游特有的交互场景（四拍扩展）

在通用四拍格式基础上，自驾游新增以下典型交互场景：

### 场景一：路线选择

用户只说了"从成都去丹巴"，有多条路线可选：

```
Re-ground:  行程规划, D1 路线选择
Simplify:   成都到丹巴有两条主要路线，差1.5小时。
Recommend:  建议走217省道线（364km/5h/¥105），比318线更快、更短、4月春花烂漫。
Options:
  A) 217省道线 — 快1.5h，春花好，路况稳（推荐）
  B) 318/夹金山线 — 经典川藏线体验，但多1.5h、路况略差
  C) 全高速绕行 — 最稳但最远，看不到省道风景
```

### 场景二：车型决策

双家庭6人出行，一车还是两车：

```
Re-ground:  行程规划, 车型选择
Simplify:   6个人出行，一车还是两车？
Recommend:  建议一辆7座MPV（格瑞维亚），省钱省心，一家出一位司机即可。
Options:
  A) 一辆7座MPV — 租车费省一半，但全家挤一辆（推荐）
  B) 两辆5座SUV — 灵活度高，但油费/过路费翻倍
  C) 先看价格再定
```

### 场景三：驾驶节奏取舍

行程塞了太多点，某天驾驶量超标：

```

Recommend:  建议砍掉德夯深度游，只走矮寨大桥观景台打卡（30分钟），保凤凰古城的游览时间。
Options:
  A) 砍德夯，只打卡矮寨大桥（推荐）
  B) 都去，但凤凰只能看夜景
  C) 德夯和凤凰都保留，晚上8点后才到凤凰
```

## 核心原则（增量）

自驾行程的核心约束是 **驾驶时长**，不是景点数量。

长途日少排景点，短途日多排体验。行程不是越满越好，是每天驾驶量合理、到达目的地后还有精力玩才算好。

路线选择是三角权衡：时间、路况、风景。亲子出行优先时间和路况；纯成人可牺牲时间换风景。

车辆是行程的骨架：取车/还车节点决定首尾天的路线，车型决定人数和舒适度，全险和儿童座椅是底线配置。
