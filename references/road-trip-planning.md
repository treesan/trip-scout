# 自驾游行程规划方法论

> 基于 `trip-planning.md` 通用方法论，叠加自驾游特有的决策维度。通用部分（四拍交互、选点/选餐原则、信息源优先级）不重复，只写自驾增量。

---

## 一、自驾游 vs 通用出行的差异清单

| 维度 | 通用出行 | 自驾游增量 |
|------|---------|-----------|
| 交通 | 地铁/公交/打车，时效可控 | 自驾，时效受路况/天气/节假日车流影响，里程和时长是硬约束 |
| 车辆 | 无 | 取车/还车节点、车型选择、儿童座椅、全险、异地还车费 |
| 路线 | 地铁线路唯一选择 | 高速/国道/省道多选，路况、区间测速、冬季封路需评估 |
| 驾驶节奏 | 步行5km算极限 | 每天驾驶不宜超4-5h（亲子）/6-7h（纯成人），长途日必须配轻量游玩 |
| 费用 | 交通费固定 | 油费/过路费/租车费/轮渡费/异地还车费，高速免费时段可利用 |
| 住宿 | 酒店位置决定区域 | 住宿必须考虑停车场（免费停车优先）、取车方便性 |
| 车队 | 单人/双人 | 双家庭多车协作：同步出发、AA分摊、通信方式 |
| 风险 | 天气/预约 | 路况（高原山路/雨雪/塌方）、夜间山路、新能源补能、轮渡停航 |

## 三、自驾游判断顺序（在通用判断顺序基础上插入）

通用判断顺序为：天数够不够 → 酒店在哪 → 用户讨厌什么 → 天气 → 预约 → 节假日 → 补餐厅

**自驾游插入额外步骤**：

1. 天数够不够
2. **取车/还车节点在哪**（决定首尾天路线）
3. **总驾驶里程是否超限**（亲子每天≤300km/4h，纯成人≤500km/6h）
4. 酒店在哪（优先免费停车、取车方便）
5. 用户讨厌什么
6. **路线选择：高速 vs 国道 vs 省道**（风景、路况、时间三角权衡）
7. 哪些点看天气
8. 哪些点需要预约
9. 哪些点会在节假日爆炸
10. **驾驶日不能排太满的景点**（长途日配轻量游玩，短途日可多排）
11. 最后才补餐厅和咖啡

## 四、路线选择原则

### 路线三角权衡

每段路线需在 **时间 / 路况 / 风景** 三角中做取舍：

| 优先级 | 典型场景 | 选择倾向 |
|--------|---------|---------|
| 时间优先 | 长途赶路日、最后一天返程 | 全高速，不绕路 |
| 风景优先 | 当天驾驶量轻、用户明确想看沿途风光 | 国道/省道（如217省道、G227最美国道） |
| 路况优先 | 亲子出行、雨雪天、夜间驾驶 | 避开省道山路，选高速或路况稳定的国道 |

### 路线评估必查项

- **区间测速**：川西、新疆等旅游区大量区间测速，必须遵守限速
- **冬季封路/塌方**：高原省道冬季可能封路，出发前查当地交通公众号
- **夜间山路**：绝不安排夜间走山路（弯道多、无路灯、动物出没）
- **炮弹坑路**：部分省道/县道路况极差（如丹巴→玉科大草原112km/3.5h炮弹坑路），亲子出行直接排除
- **服务区质量**：热门线路服务区可能爆满（天全服务区节假日不建议吃饭）

### 路线数据来源

- **高德地图驾车路线规划API**：里程、时长、过路费逐段校正
- **当地交通公众号/微博**：封路、塌方实时通知
- **小红书近期游记**：路况实况、测速提醒

## 五、驾驶节奏规划

### 每日驾驶上限

| 人群 | 每天最大驾驶时长 | 每天最大里程 | 说明 |
|------|-----------------|-------------|------|
| 亲子（含6岁儿童） | ≤4-5小时 | ≤300km | 儿童在车上超过5h会烦躁 |
| 纯成人 | ≤6-7小时 | ≤500km | 超过7h疲劳驾驶风险急剧上升 |
| 高原山路 | 降低30% | 降低30% | 弯道多、海拔高、注意力消耗大 |

### 驾驶日配景点原则

- **长途日（≥3h驾驶）**：只配1个轻量景点 + 1顿顺路餐，当天核心任务是到达目的地
- **短途日（≤1.5h驾驶）**：可排2-3个景点 + 自由时间
- **纯驾驶日**：只排路线亮点（沿途观景台、服务区美食），不排正式景点
- **第一天**：取车 + 轻量适应（≤2h驾驶），不跑远
- **最后一天**：还车 + 返程，不排景点（除非还车点就在景区附近）

### 出发时间偏好（亲子）

- 亲子自驾默认 **9:30出发**，拒绝早起
- 唯一例外：核心景点需要开门就进（如七彩丹霞看晨光），可安排7:30出发但前一晚必须早睡
- 每天写清"几点出发"，不做模糊表述

## 六、车辆与租车规划

### 车型选择优先级

| 场景 | 推荐车型 | 原因 |
|------|---------|------|
| 双家庭6人 | 7座MPV（丰田格瑞维亚/岚图梦想家） | 两家坐一车省钱省心，空间大儿童友好 |
| 双家庭多车 | 各开一辆5座SUV | 灵活度高，AA分摊简单，但油费翻倍 |
| 3-4人 | 5座SUV或轿车 | 够用，油耗低 |
| 高原/山路 | SUV（底盘高、抓地力强） | 省道县道路况保障 |

### 租车关键项

- **全险必买**：自驾游路况不可控，磕碰概率高
- **儿童安全座椅**：租车平台可租（一嗨/神州），提前APP锁定+电话确认门店库存
- **异地还车费**：计算进总预算（如海口取→湛江还，一嗨3995元含异地费）
- **油量规则**：满油取满油还（提前在还车点附近加满油）
- **车型锁定**：提前3天APP锁定+电话确认，避免到店无车

### 车队协作（双家庭）

- **出发同步**：约定统一出发时间，途中保持微信/电话联络
- **AA分摊**：租车费、油费、过路费按家庭分摊；餐饮/门票按人头
- **轮渡等特殊场景**：提前统一购票，升舱统一决策（五星船咖啡厅200元/6人）
- **导航方式**：各车独立导航，不依赖跟车（避免走错路口掉队）

## 八、路况风险评估

### 风险矩阵模板

| 风险 | 概率 | 影响 | 应对方案 |
|------|------|------|---------|
| 高原山路雨雪 | 中-高 | 驾驶危险+延误 | 雨雪天改走高速避山路；备防滑链 |
| 区间测速 | 高 | 罚款+扣分 | 严格遵守限速，导航提示测速点 |
| 夜间山路 | 低-中 | 安全风险 | 绝不安排；如被迫则就近找住宿不赶路 |
| 省道塌方/封路 | 低 | 延误或改道 | 出发前查当地交通公众号；备备选路线 |
| 轮渡停航（大雾） | 低-中 | 整天行程报废 | 提前购票；备改签方案；不排紧凑衔接 |
| 租车无车 | 低 | 出行受阻 | 提前3天锁定+电话确认 |
| 新能源补能困难 | 中 | 续航焦虑 | 混动优先（加油即可）；纯电查沿途充电桩密度 |

## 九、住宿选择增量约束

自驾住宿除了通用考量（位置、品质），追加：

- **停车场**：必须有，免费优先（市区酒店停车场可能¥50-100/晚）
- **取车方便**：如租车，酒店离取车/还车点越近越好
- **车位宽度**：7座MPV/SUV需要更宽车位，老城区小酒店可能进不去
- **充电桩**：新能源需确认酒店是否有充电设施

## 十二、自驾游常见坑（在通用常见坑基础上追加）

| 常见坑 | 说明 |
|--------|------|
| 不看每日驾驶上限 | 亲子一天开6h+，孩子崩溃大人疲惫 |
| 安排夜间山路 | 弯道+无灯+动物=极度危险 |
| 不查区间测速 | 旅游区测速密集，罚款+扣分 |
| 不估算油费/过路费 | 自驾费用大头，低估导致预算爆表 |
| 不确认停车场 | 市区老酒店无车位，7座MPV进不去 |
| 忽略高速免费时段 | 五一/国庆/清明/春节高速免费，可省大量过路费 |
| 不提前锁定车型 | 到店发现没车，被迫换车型或延误 |
| 不备离线地图 | 高原/山区信号差，导航断线 |
| 不看轮渡/航班衔接 | 轮渡停航或航班延误导致整天行程报废 |
| 把长途驾驶日排满景点 | 长途日只能配轻量活动，否则赶不到目的地 |
| 不标路线路况 | 用户不知道某段是山路/省道/测速路段，驾驶时措手不及 |

## 十三、自驾游特有的交互场景（四拍扩展）

### 场景一：路线选择

用户只说了"从成都去丹巴"，有多条路线可选：

```
Re-ground:  行程规划, D1 路线选择
Simplify:   成都到丹巴有两条主要路线，差1.5小时。
Recommend:  建议走217省道线（364km/5h/¥105），比318线更快、更短、4月春花烂漫。
Options:
  A) 217省道线 — 快1.5h，春花好，路况稳（推荐）
  B) 318/夹金山线 — 经典川藏线体验，但多1.5h、路况略差
  C) 全高速绕行 — 最稳但最远，看不到省道风景
```

### 场景二：车型决策

双家庭6人出行，一车还是两车：

```
Re-ground:  行程规划, 车型选择
Simplify:   6个人出行，一车还是两车？
Recommend:  建议一辆7座MPV（格瑞维亚），省钱省心，一家出一位司机即可。
Options:
  A) 一辆7座MPV — 租车费省一半，但全家挤一辆（推荐）
  B) 两辆5座SUV — 灵活度高，但油费/过路费翻倍
  C) 先看价格再定
```

### 场景三：驾驶节奏取舍

行程塞了太多点，某天驾驶量超标：

```

Recommend:  建议砍掉德夯深度游，只走矮寨大桥观景台打卡（30分钟），保凤凰古城的游览时间。
Options:
  A) 砍德夯，只打卡矮寨大桥（推荐）
  B) 都去，但凤凰只能看夜景
  C) 德夯和凤凰都保留，晚上8点后才到凤凰
```

## 核心原则（增量）

自驾行程的核心约束是 **驾驶时长**，不是景点数量。

长途日少排景点，短途日多排体验。行程不是越满越好，是每天驾驶量合理、到达目的地后还有精力玩才算好。

路线选择是三角权衡：时间、路况、风景。亲子出行优先时间和路况；纯成人可牺牲时间换风景。

车辆是行程的骨架：取车/还车节点决定首尾天的路线，车型决定人数和舒适度，全险和儿童座椅是底线配置。
