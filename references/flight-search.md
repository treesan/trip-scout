# 航班搜索工作流

## 触发条件

用户提到机票、航班、出发/到达相关关键词：
- "搜一下 7月15号 上海到杭州的机票"
- "查一下明天成都飞北京的航班"
- "flight search Beijing to Shanghai"

## 输入提取

从用户输入中提取：

| 字段 | 说明 | 必填 |
|------|------|------|
| origin | 出发城市/机场 | ✅ |
| destination | 目的地城市/机场 | ✅ |
| date | 出发日期 | ✅ |
| back_date | 返程日期 | ❌ |
| passengers | 人数 | ❌ (默认2大人) |
| direct_only | 是否只看直飞 | ❌ |
| date_range_start | 日期范围起点（机酒联动用） | ❌ |
| date_range_end | 日期范围终点（机酒联动用） | ❌ |

## 机场信息输出规则（强制）

航班搜索结果**必须输出出发和到达机场全名**，不能只输出城市名。原因：

- 同一城市可能有多个机场，距离市区差异极大，直接影响：
  1. **自驾取车**：机场距租车门店/市区酒店的距离决定D1行程节奏
  2. **酒店选择**：机场远近决定是否需要在机场附近住宿
  3. **出行时间**：天府机场等远郊机场需提前1.5-2h出发

**多机场城市示例**：

| 城市 | 机场 | 距市区 | 特点 |
|------|------|--------|------|
| 成都 | 双流国际机场(CTU) | ~15km/30min | ✅ 离市区近，优先 |
| 成都 | 天府国际机场(TFU) | ~50km/1.5h | ❌ 非常远，除非票价显著便宜 |
| 上海 | 虹桥国际机场(SHA) | ~15km/40min | ✅ 离市区近 |
| 上海 | 浦东国际机场(PVG) | ~40km/1h | 偏远 |
| 北京 | 大兴国际机场(PKX) | ~50km/1h | 偏远 |
| 北京 | 首都国际机场(PEK) | ~25km/40min | 相对近 |

**比价规则**：同城市近郊机场（如双流）比远郊机场（如天府）贵≤¥100/人时，**优先选近郊机场**。¥100换1.5小时通勤时间+租车便利，完全值得。

## 机酒联动日期优化（场景二专用）

当用户给出日期范围（如"9月25日到10月7日之间选9天"），不能直接定出发日期，需在范围内搜索多条日期组合的机酒总费用，选出最优。

### 假期日历（辅助参考，非必须）

> **核心原则**：只要能获取到机票和酒店出行前后几天的真实价格，就能算出最优出发日。假期日历只是辅助判断价格拐点，不影响计算本身。

**查询方式**：
- 搜索引擎查询"万年历"或"XX年放假安排"（如已安装 baidu-search 技能，可直接搜索）
- 在线查询：百度搜索"万年历" → 获取法定假日+调休信息

**用途**：预判价格拐点（假期前1天涨价、假期最后1-2天回落），缩小搜索日期范围。如果已拿到逐日价格数据，假期日历不是必须的。

**关键假期**（影响出行价格）：

| 假期 | 2026日期 | 天数 | 价格影响 |
|------|---------|------|---------|
| 元旦 | 1.1-1.3 | 3天 | 机票+20-30%，酒店+10-20% |
| 春节 | 2.17-2.23 | 7天 | 机票+50-100%，酒店+100-200% |
| 清明 | 4.4-4.6 | 3天 | 机票+20-30%，酒店+10-20% |
| 劳动节 | 5.1-5.5 | 5天 | 机票+30-50%，酒店+30-50% |
| 端午 | 5.31-6.2 | 3天 | 机票+20-30%，酒店+10-20% |
| **中秋节** | **9.25-9.27** | **3天** | **机票+20-40%，景区酒店+20-30%** |
| **国庆节** | **10.1-10.7** | **7天** | **机票+50-100%，景区酒店+50-200%** |

**请假凑假攻略**（用户常见需求）：
- 中秋+国庆: 请9.28(周一)-9.30(周三)3天 → 连休13天(9.25-10.7)
- 劳动节: 请4.28-4.30 3天 → 连休9天(4.26-5.5)

**假期对价格的影响规律**：
- 机票：假期首末日价格最高，中间日相对低；假期后1-2天价格断崖式回落
- 酒店：景区酒店假期溢价远大于城市酒店；假期前1天开始涨价，假期最后1-2天回落
- **景区酒店国庆溢价实例(2026-07-08携程真实数据)**：
  - 赛里木湖亚朵见野: 平日¥517 → 国庆¥757 (+46%)
  - 伊宁福朋喜来登: 平日¥262 → 国庆¥411 (+57%)
  - 乌鲁木齐万达锦华: 平日¥238 → 国庆¥238 (0%，城市酒店无溢价)
  - 那拉提英迪格: 平日¥554 → 国庆¥499 (-10%，景区淡季清仓降价)

### 搜索策略

1. **获取机票逐日价格** — 对候选出发日期，搜索去程+回程机票价格（飞猪API），获取真实价格而非估算
2. **获取酒店逐日价格** — 对每个住宿点hotelId，获取出行日期范围内的每日房价（见下方"酒店每日价格获取"），**这是核心数据**
3. 确定可选出发日期范围（用户指定范围内，排除不合理的组合）
4. 计算每条日期组合的机酒总费用（机票+住宿），输出最优出发日期 + 其他组合对比
5. 假期日历仅辅助预判价格拐点、缩小搜索范围，**有了真实价格数据则不需要**

### 酒店每日价格获取（机酒联动关键数据）

**方法1：携程浏览器模式（推荐， 已验证可行）**

用Playwright打开携程酒店详情页，从页面DOM提取价格。每个hotelId×每个日期组合访问一次详情页。

```python
# 伪代码: 批量获取酒店每日价格
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    context = browser.new_context(user_agent="Chrome/120...")
    # 加载携程Cookie (脚本默认存储位置, 见 scripts/ctrip_reviews.py login)
    # Cookie path 由 ctrip client 模块管理 (默认 ~/.ctrip/cookies.json, 可通过脚本配置)
    page = context.new_page()
    
    for hotel_id in hotel_ids:
        for checkin, checkout in date_pairs:
            url = f"https://hotels.ctrip.com/hotels/detail/?hotelId={hotel_id}&checkIn={checkin}&checkOut={checkout}"
            page.goto(url, wait_until='domcontentloaded', timeout=15000)
            # 提取页面中所有¥价格
            prices = page.evaluate(r"""() => {
                const body = document.body.innerText;
                const prices = [];
                const re = /¥([\d,]{3,5})/g;
                let m;
                while ((m = re.exec(body)) !== null) {
                    const val = parseInt(m[1].replace(/,/g, ''));
                    if (val >= 200 && val <= 9999) prices.push(val);
                }
                prices.sort((a, b) => a - b);
                return [...new Set(prices)].slice(0, 10);
            }""")
```

## 双平台并行查询

### 飞猪 (flyai)

```bash
flyai search-flight \
  --origin "上海" \
  --destination "杭州" \
  --dep-date 2026-07-15 \
  --sort-type 3
```

关键参数：
- `--sort-type 3` = 价格升序
- `--journey-type 1` = 直飞
- `--max-price` = 价格上限

返回格式：JSON，包含航班号、时间、价格、航司等

### 携程 (wendao)

```bash
jq -n \
  --arg token "$WENDAO_API_KEY" \
  --arg query "搜索7月15号上海到杭州的航班" \
  '{token: $token, query: $query}' | \
  curl -s -X POST https://wendao-skill-prod.ctrip.com/skill/query \
  -H "Content-Type: application/json" \
  -d @- > /tmp/wendao-flight.json
```

查询模板：
- "搜索{日期}{出发}到{目的地}的航班"
- "搜索{日期}{出发}到{目的地}的直飞航班"
- "搜索{日期}{出发}到{目的地}的航班，经济舱"

返回格式：Markdown 正文，需提取航班信息

## 结果合并与排序

1. 从两个平台提取航班信息
2. 按航班号匹配（同一航班对比价格）
3. 按价格升序排列
4. 标注：直飞/转飞、航司、是否有餐、行李额

## 错误处理

- 飞猪无结果 → 只展示携程结果，标注"飞猪未找到"
- 携程 API 超时 → 重试1次，仍失败则只展示飞猪结果
- 两个都失败 → 提示用户稍后重试或手动查询
